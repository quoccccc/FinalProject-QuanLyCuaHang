# Quản lý cửa hàng tạp hóa

## Công nghệ sử dụng
- C# WinForms
- Entity Framework Core (Code First)
- SQL Server
- Mô hình 3 lớp (Entity – DAL – BLL – UI)

## Chức năng
- Quản lý sản phẩm
- Bán hàng
- Theo dõi tồn kho
- Báo cáo doanh thu

## Hướng dẫn chạy
1. Clone project
2. Update-Database
3. Run project
